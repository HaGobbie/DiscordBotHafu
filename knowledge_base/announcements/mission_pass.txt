--- [Section: 概要] ---
Implemented on October 13, 2021 Content that allows you to earn rewards by collecting "stars" that you receive by completing specific daily tasks and weekly tasks. You can check the contents from Main Menu → Tasks/Quests → Mission Pass or by using the shortcut keys listed under Tasks. Mission pass rewards are not automatically earned. Please note that all items must be obtained manually from this screen. (In the 2023/10/18 update, the "Get All" button was implemented, allowing you to receive all at once, including the gold reward frame.) Every time you collect 10 stars, the "tier" level increases, and you can earn rewards according to the tier you reach. The number of tiers is 40 tiers. The starting tier is 1, so you will need 390 stars to get all the rewards. You can also purchase tiers with star gems from the mission pass screen (1 tier: 25SG). Approximate number of stars that can be obtained in a month Daily tasks 10 stars x 30 days 300 pieces Weekly tasks 90 stars x 4 weeks 360 pieces Total 660 pieces The content of distribution will change depending on the "Season" update, and the stars and tiers collected will be reset at the same time as the update. Costumes, accessories, stamps, magforms, and SG scratch prize exchange tickets will be updated as major rewards. Other reward items will basically remain the same, with only updates to the contents of material items and special ability capsules. You can only receive rewards from the previous season. Switch to "Previous Season" in the Mission Pass window. ``NGS'' mission pass rewards tend to be reprinted (redistributed) through limited-time special scratches. *1 In the past, systems such as "overrun" and "group compensation" existed, but both have been discontinued. ▼ Change history 2023/10/18 (Season 25) Adjusted the gold reward lineup so that you can receive all tier rewards at once (group reward system completely abolished) Among the rewards, cosmetics/parts have also been adjusted so that you can get 2 items at a time 2023/08/09 (Season 23) Enhanced the gold reward lineup. --- [Section: ゴールド報酬] ---
To obtain the "Gold Rewards" displayed at the bottom of the Mission Pass, you need the "N-Mission Gold Pass". Use the pass to earn all gold rewards for the season, including any tiers you reached before using it. Rare Item Text How to Obtain 5 N-Mission Gold Pass With the Mission Pass, you will be able to obtain gold rewards in addition to regular rewards only during the current season. SG Shop (100SG) * *: If you try to receive gold rewards without using the pass, a purchase confirmation window will be displayed. In this case the purchase pass is automatically used. During the premium set usage period, the N-Mission Gold Pass will automatically be used for the currently distributed season. Even if the Premium Set usage period ends in the middle, the pass will remain in use until the end of the receiving period. Please note that even if you activate the premium set after using the pass, there will be no refunds and it will be wasted. Only the current season can be unlocked by starting a Premium Set or using a Gold Pass from an item pack. If you want to release an unopened slot from the previous season, you will need to switch the display to the previous season, attempt to receive the gold reward, and display the pass purchase confirmation. ▲ ▼
