--- [Section: 概要] ---
You can check the task list by going to Main Menu → Tasks/Quests → Tasks, or pressing the [R] key or X button while displaying Simple Status. By meeting the achievement conditions, you can advance the story, unlock features, and receive rewards. The effect of completing a task will occur automatically upon completion, and you will receive the reward on the spot. Therefore, it is not possible to achieve the goal and receive the reward using a different character. It corresponds to the old PSO2's Client Order and Arks Mission. Main Task An important task that serves as a substitute for story progression or a tutorial. It is a condition for unlocking some sections and high-rank sections. Side Task A task from an NPC other than the main task. There are a wide variety of activities such as defeating designated enemies, collecting them, and exploring them, and many of them reward you with a large amount of experience points. Daily/Weekly Tasks Tasks that are not accepted from a specific NPC, but are automatically accepted when you log in. In order to receive an order, it is necessary to make some progress in the main task. Immediate completion reward. Be careful of ships and characters when receiving rewards. Limited time task A task that occurs only during an event, etc. for a limited time. Please note that there is a set period in which you can accept orders, and you will no longer be able to receive or report orders after that period. ▼ Change history 2023/01/25 Update When displaying the task guide, the specifications have been changed so that limited-time tasks and daily tasks are displayed preferentially in the guide.

--- [Section: ガイドマーカー] ---
The task has a guide marker function that indicates the location of the target point, so even beginners can reach the target point without getting lost. If you accept multiple tasks, the guide marker for the last task you accepted will be automatically displayed, but you can change the task for which the guide marker will be displayed from the task list displayed by selecting ``Tasks/Quests'' from the menu.

--- [Section: 達成済みタスクの確認方法] ---
You can check by changing "In progress" to "Completed" at the top of the task list.

--- [Section: サイド] ---
Clicking on the requested NPC will take you to the task details for each NPC. For some tasks, such as Trainier's Conqueror, if you accept the task after fulfilling the conditions, you will be able to hear special lines. The maximum number of unfulfilled orders is 20, which increases to 40 if you use the premium set. Request NPC Features Common to all regions Riamay Tasks that can be ordered every day, some weekly tasks Ronaldin Loop tasks for advanced users Tim Team tasks For limited tasks that can be accepted only once, refer to each NPC in NPCs.

--- [Section: デイリー] ---
*Only once per day per account. Unlocked by completing ``Talk to Aina at the Rest Corner'' in the main task ``Preparing for Battle.'' A task that can be ordered once a day. A total of five tasks will be selected, one from each genre. The content changes will occur every day at 4am, similar to the Gathering. Although it can be said that orders are almost automatically accepted, you will need to "log in," "move to a room," or "view the task list." In other words, if you cross 4 o'clock while playing, you will need to consciously re-order. If you forget to order a task, you may fail to complete the daily task after the content has been updated. (For example, when you realize that you forgot to update after completing a section-specified gathering, etc.) Players who are active during this time should get into the habit of opening the task list. In the opposite case, if you have tasks left for the previous day, if you can accomplish them in that room, you can complete them and receive the reward before retaking the order. You can also use this method to follow up on cases where you failed to achieve the goal due to forgetting to update, as mentioned earlier. (You need to wake up around 4 o'clock...) The contents of the task will change depending on the progress of the main task, and the task corresponding to the character whose main task is the most progressed in the account will be issued. If you proceed with a low-level character after high-level tasks have been issued, be aware that while you will gain a lot of experience points if you complete them, there are also tasks that will become virtually impossible to progress. Guide marker function is also effective.
