## Strategy
Quests that belong to Duel Quest are divided into layers such as ``Phase'' and ``Depth.'' Here, we will describe common strategy information across each level.

## standard duel
[About clear/failure conditions/inability to fight] The clear condition for each is "defeating the target enemy." The quest will fail after the 10 minute time limit has passed. A one-on-one battle will take place against a boss enemy with the double name "Indomitable." If you are unable to fight, you will not be able to use "Revival on the spot" after a certain period of time has passed, and it will fail unless you use "N-Scape Doll" or "N-Half Doll". [About the enemy with the double name "Unyielding"] In Standard Duel, the following correction multiplier is applied to various parameters of the enemy with the name "Unyielding". In particular, the power of attacks against players is 5 times greater, so you must be careful. List of parameter corrections for appearing enemies (click to expand) Appearing quest Rank. time -20% "Defi" type special abilities do not conflict unlike "Soul" type or "Gigas" type, so multiple types of "Defi" type special abilities can be added to one equipment. "Defi" type special abilities are effective only when using weapons that can be equipped by the main class. Determined by whether or not you are using a weapon for your main class. For multi-weapon weapons, the effect will only be applied while the weapon type for the main class is being used. The same judgment applies to the armor side, as long as the type of weapon being used matches. There is no problem even if the weapon does not have the "defi" type special ability, so if you attach it to the armor first, it will be easier to use it for each class. However, be aware that you will inevitably not be able to fight with subclass weapons, as they will all become invalid the moment you switch to a non-main class weapon. The power increase effect per "defi" type special ability is extremely high at +7%, so if you aim to clear the battle consistently, specialized equipment is almost essential. Adding "Defi Haumes P.01 ~ P.05", "Defi Elisis ~ P.05", and "Defi Sayres ~ P.05", which can be purchased cheaply from My Shop, to all weapons and armor is enough. In particular, attaching "Defi Elisis P.01 to P.05" (damage resistance +20% to 30%) in four locations will prevent instant death from non-major moves, so give it priority.
