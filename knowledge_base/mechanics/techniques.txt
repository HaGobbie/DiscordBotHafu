The setting of the Phantasy Star series is that ``magic'' requires innate talent, and ``technique'' is the reproduction of magic using science and technology. Techniques have six attributes: fire, ice, lightning, wind, light, and darkness, and each technique behaves very differently. Note that the classes and weapon types that can be used with complex attribute techniques are different from regular techniques, and it is also necessary to learn specific class skills. To use a technique (excluding complex attribute techniques described below), you must set a class that can use the technique as your main class or subclass, and equip a weapon type that can use the technique. Usable Classes Force Tector Bouncer Usable Weapons Rod Talis Wand Jet Boots Technique attacks have the following characteristics. By charging (holding down the button), the power and behavior change. Attacks have attributes (weapon attacks are non-attribute, except for some weapons with attributes). Attacks with an attribute that matches the enemy's weak point attribute will increase damage. By attacking an enemy with a weak point attribute technique, you can lower the attribute. ▲ ▼ Mixed Attribute Techniques Combined Attribute Techniques can only be used with Force and Techter weapons. It is also necessary to learn the class skill "Unite Technique Fo" or "Unite Technique Te". Usable Class: Force Tector Usable Weapon Type: Rod Talis Wand Unlike normal techniques, it consumes a "composite attribute technique gauge" instead of PP. There is no damage increase due to matching charge or weak point attributes, and class skills that target techniques are not applied.
