--- [Section: 概要] ---
Techniques are like magic. The setting of the Phantasy Star series is that ``magic'' requires innate talent, and ``technique'' is the reproduction of magic using science and technology. Techniques have six attributes: fire, ice, lightning, wind, light, and darkness, and each technique behaves very differently. Note that the classes and weapon types that can be used with complex attribute techniques are different from regular techniques, and it is also necessary to learn specific class skills. To use a technique (excluding complex attribute techniques described below), you must set a class that can use the technique as your main class or subclass, and equip a weapon type that can use the technique. Usable Classes Force Tector Bouncer Usable Weapons Rod Talis Wand Jet Boots Technique attacks have the following characteristics. Power and behavior change by charging (holding down the button) Attacks have attributes (weapon attacks have no attributes except for some weapons with attributes) Attacks with attributes that match the enemy's weak point attribute will increase damage You can "lower attributes" by attacking enemies with weak attribute techniques ▲ ▼

--- [Section: 複合属性テクニック] ---
Multi-attribute techniques can only be used with Force and Tector weapons. It is also necessary to learn the class skill "Unite Technique Fo" or "Unite Technique Te". Usable Class: Force Tector Usable Weapon Type: Rod Talis Wand Unlike normal techniques, it consumes a "composite attribute technique gauge" instead of PP. There is no damage increase due to matching charge or weak point attributes, and class skills that target techniques are not applied. See Techniques/Complex Attributes for details. ▲ ▼
