## Add-on skill generation
To generate it, you need N-Class Excube or N-Master Cube and N-Meseta corresponding to each class. After generating a certain number of times (regardless of class), the higher generation method will be released. The higher the generation method, the higher the minimum level of the main effect and sub effect. Although the exact lottery probabilities for each of skill generation 1 to 3 need to be verified, the idea is that ``if the lottery result is below the lower limit level, it will be overwritten.'' Therefore, even if you select skill generation 3, it does not mean that it will be easier to obtain something of level 11 or higher. ## main effect
The contents of the main effect are fixed for each class. A random level is generated each time an add-on skill is generated, but it will never be lower than the level of the previously generated skill. For example, once you generate a level 13 main effect, the level of the main effect will be 13 or higher regardless of the generation method. The image of the lower limit level in generation is ``If the lottery result is lower than the highest level in the past, it will be overwritten,'' and it seems that even if the lower limit level increases, it will not become easier to obtain high-level items. Therefore, if the purpose is to raise the level of the main effect, skill generation 2 to 5 only needs to be done the first time.

## sub effect
There are multiple sub effects for each class, and two types are selected from among them when creating add-on skills. The content and level will be random, but the lower limit of the level will be fixed depending on the generation method. If you want to aim for Lv.11 or above for both sub effects, N-Meseta will be more efficient if you carefully select it in skill generation 1. Some sub-effects have a class name abbreviation (Xx) at the end of their name, but this does not mean that the effect is limited to that class; it applies to all classes if set. If the effect is of the same type, the effect will be applied multiple times. If you set "HP Up (Xx)" for all classes to Lv.20, 1 class HP20 x 10 classes = maximum HP200 will be applied. If you set "HP/PP recovery when participating in a trial (Xx)" for all classes to Lv.20, HP will recover up to 100% of the gauge and PP will recover up to 100% of the gauge.

## Set of add-on skills
One main effect and two sub effects can be registered for one set.

## My set: Add-on skills
Just like "My Set", which stores class and equipment information, you can also save and register add-on skill set information. There are three set frames for each class to store add-on skill combinations.

## addon skill boost
Gained experience points, acquired N-Meseta, and rare drop multipliers will each increase by 1% x the number of classes that generated add-on skills. As of April 5, 2023, the upper limit is 10%. The effect occurs when the add-on skill is generated. It becomes effective even if you select a set with no registered effects.

## Digression
According to Schrafft, the Arks Comprehensive Tactics Instructor, add-on skills are "new skills that are generated using 'N-Class Ex-Cube' as a reaction agent." He also said of the N-Class Ex-Cube, ``It can be acquired by gaining a lot of experience,'' and ``I don't know the exact principle, but it is said that the photons within the person crystallize as they grow.'' This system strengthens a character's basic abilities by unlocking additional skills prepared for each class. The main way to obtain the ``N-Class Excube'' needed to generate add-on skills is to ``obtain more experience points after reaching the level cap'' and skill generation includes random elements, so some players may be reminded of elements that appeared in the following series works. In the old PSO2, systems such as ``Timed Ability Installation'', which creates items that give special effects to armor (units), and ``Class Boost'', which allows you to obtain permanent/account-shared strengthening effects as a reward for the title of ``reaching a certain level of a specific class'', were introduced. Additionally, equipment called ``Skill Rings'' has also appeared that (with some conditions) allows the equipped character to learn various skills. In the previous game ``PSU'', a system called ``GAS (Guardians Advanced Style)'' was introduced that strengthens characters using special experience points obtained when the level cap is reached.
