# Quests Overview — All Quest Types and How to Access (Part 5)

== Time Extension Quests ==
Time Extension Quests are instanced quests where players must kill waves of enemies to extend the timer to avoid timing out before defeating the final boss.
Currently, the available Time Extension Quests alternate between the Retem region Time Extension Quests and the Kvaris region Time Extension Quests daily.
