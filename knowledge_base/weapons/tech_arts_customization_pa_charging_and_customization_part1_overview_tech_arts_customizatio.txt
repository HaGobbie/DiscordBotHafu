# Tech Arts Customization — PA Charging and Customization System (Part 1)

Sections in this file: Overview, Tech Arts Customization Costs

== Overview ==
A system that changes the performance of Photon Arts and Techniques. You can unlock, upgrade, and set customization effects at the Class Counter. This system is locked until the character completes Main Story Chapter 6: "Starless", A New Enemy.
There are three different customization effects for each Photon Art or Technique that can be generated using materials items and N-Meseta. Each level of an effect adds or increases a positive effect, lowers a negative effect or both.
Each customization effect is saved separately and is freely changeable at the class counter without losing levels. Note that classes that share weapons or can use techniques share the set customization and level. For example if you unlock Technique "Grants" as Type 1 and upgrade it to Level 5, this is shared with all other classes that can use techniques. The same applies to Photon Arts.
