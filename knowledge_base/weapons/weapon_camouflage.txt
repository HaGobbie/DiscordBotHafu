--- [Section: 概要] ---
Weapon camouflage is an item that only changes the appearance of a weapon. You can set it for each weapon type by going to Main Menu → Equipment/Sub Palette → Weapon Camouflage. Weapon camouflage item names always have an asterisk (*) at the beginning. Some weapon camouflages are compatible with multiple weapon types. Old PSO2 weapon camouflage can also be used. Weapon camouflage implemented in NGS cannot be used in old PSO2. Even if the graphics are from the old PSO2, if they are handled as NGS specifications, they will be NGS-only. Currently, if any of the following apply, the weapon camouflage will be NGS specification. The flavor text of the item details is ``Weapon camouflage that changes the appearance of equipment.'' The item names are ``＊○○○ Almaty,'' ``＊○○○ Tayou,'' and ``＊Clio ○○○.'' ▲ ▼

--- [Section: カラーバリアント] ---
When using equipment with a color variant set, the priority of the effect color is in the order of "Weapon camouflage > Weapon > Default color (blue)". ▼ Priority example Basic: Prim + Tsuvia camouflage → Since neither is CV, the effect color will be the default blue. Example 1: Prim + Tsuvia camouflage CV → Since only camouflage is a CV, the effect color will be the green of the Tsuvia camouflage CV. Example 2: Prim CV + Tsuvia camouflage → Only weapons are CV, so the effect color will be the yellow of Prim CV. Example 3: Prim CV + Tsuvia camouflage CV. →Both are CVs, but the camouflage one takes priority, so the effect color will be the green of the Zvia camouflage CV.Camouflage colors that have a color variant cannot be changed. Also, the color variant of weapon camouflage is only compatible with the one set for multi-weapons. In other words, if you set only one side, different colors will appear on the base side and the material side. ▼ Priority example when using multi-weapon Example 1: Set Prim Sword/Pulchi + Tsuvia camouflage CV to both sort and Parti → In both sort and Parti, the effect color will be the green of Tsuvia camouflage CV Example 2: Set Prim Sword/Parti + Tuvia camouflage CV to sword → When using Sword, the effect color will be the green of Tsuvia camouflage CV, and when using Parti, the default blue will be Example 3: Setting Prim Sword/Parti + Tsuvia camouflage CV to Parti → When using Sword, the default is blue, and when using Parti, it becomes the green of Tsuvia camouflage CV. Example 4: Setting Prim Sword/Parti + Tsuvia camouflage normal version to Sword, and Tuvia camouflage CV to Parti → When using Sword, the default is blue, and when using Parti, it becomes the green of Tsuvia camouflage CV. Example 5: Set the Prim Sword/Palchi CV + Tsuvia camouflage regular version to the sword, and set the Tsuvia camouflage CV to Parti → When using a sword, it will be the yellow of Prim CV, and when using Parti, it will be the green of Tsuvia camouflage CV ▼ Adjustment / modification history 2022/6/8 Changed the specifications so that the effect color of the weapon is displayed even when weapon camouflage is set for color variant weapons. ▲ ▼

--- [Section: フォトンカラー変更] ---
Some weapon camouflages in the old PSO2 were compatible with photon color changes, but there is no way to change them in NGS. It needs to be changed in the item lab of the "PSO2" block. (The “*Kazarodou” and “*Black Bals” distributed in the NGS new registration campaign were also implemented in the past in the old PSO2, and are compatible with photon color changes.) ▲ ▼
