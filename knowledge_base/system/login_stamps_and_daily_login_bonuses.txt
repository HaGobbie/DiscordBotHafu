# Login Stamps and Daily Login Bonuses

== Overview ==
Login Stamps is a system where logging in daily will net you rewards. You will be rewarded once a day when you log in or when you move to one of the main cities. The login stamp occurs at 7 AM PST / 10 AM EST. You can only receive one login bonus per account. The login bonus information can be found in the Main Menu -> System -> Login Bonus.
If you have an active Premium Set, you will be rewarded with additional Premium stamps in addition to the regular stamps. Even if you start using the Premium Set after receiving the regular login bonus, you will still receive the premium additional quota reward for the day when you use the ticket or move to a new section.
With the start of NGS, the old PSO2 login stamps have been discontinued.
For a list of all previous Login Stamps, please visit the Login Stamps Archive page.
