Main class ▲ ▼ Subclass Setting a subclass has the following effects. Can equip the subclass's weapon types Can use the subclass's PA/techniques Can use the subclass's class skills, excluding main class-specific skills (some skills are less effective than the main class) Until the subclass's level is 5 levels lower than the current level cap, 90% of the experience points gained by the main class will also be applied to the subclass The main class correction (10% power increase when using the main class's weapon type) will not be applied to the subclass's weapon type The status of the subclass is not reflected at all. The skill points used in the subclass are also used to calculate the combat power. ▲ ▼ EX Style Scheduled to be implemented on October 1, 2025 A system similar to classes where you select and set one from three: "Adrus", "Blitz", and "Celeste". Skills will be released depending on the style you set, and you will be able to use the special special move "After Blast". See the EX Style page for details. ▲ ▼ Class Skills Various class skills are set for each class, and can be acquired by consuming "Skill Points (SP)". Skill points can be earned by clearing cocoons and towers. For the current number of skill points that can be earned, see Obtaining Skill Points on the Trainier page. The acquired skill points are common to all classes, and each point acquired will add 1 point to all classes. For details on class skills, see each class page. With the update on June 7, 2023, you can now freely reassign skill points. ▼ Specifications before June 7, 2023 Specifications before June 7, 2023 To reassign skill points, it was necessary to do the following: Use "N-Initialize all skill trees" Add a skill tree with AC and transfer the skill tree with SG Add skill tree Class counter: 500AC Add a skill tree to a specific class. Skill tree transfer Class counter: 100SG Move the added tree to another class. Transferring is only possible for classes that have two or more trees, and it is not possible to make a specific class without a tree. Points will be reset for the relocated tree. If Tree 1 and Tree 2 exist and you transfer Tree 1 to another class, Tree 2 will be moved up and become Tree 1. In-game notation regarding class skills Some class skills have certain conditions such as class composition and weapon type used in order to exert their effects. For example, if there is a description such as "main class/sword only", this indicates that "the skill is valid only if the class is set as the main class and a sword is used." *Please note that in the above case, "/" has the meaning of "AND", not "OR". Class skills with an icon that says "SP0" can be learned without consuming skill points. An example is the Force or Tector class skill Unite Technique. ▲ ▼ Resta Sign Recovers your HP by 50% of its maximum value.
