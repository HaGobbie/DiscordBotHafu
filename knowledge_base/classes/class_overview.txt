## subclass
Setting a subclass has the following effects. Can equip the subclass's weapon types Can use the subclass's PA/techniques Can use the subclass's class skills, excluding main class-specific skills (some skills are less effective than the main class) Until the subclass's level is 5 levels lower than the current level cap, 90% of the experience points gained by the main class will also be applied to the subclass The main class correction (10% power increase when using the main class's weapon type) will not be applied to the subclass's weapon type The status of the subclass is not reflected at all.Skill points used in the subclass are also used to calculate combat power.

## EX style
Scheduled to be implemented on October 1, 2025 A system similar to a class where you select and set one from three: "Adoras", "Blitz", and "Celeste". Skills will be released depending on the style you set, and you will be able to use the special special move "After Blast". See the EX Style page for details.

## class skills
Various class skills are set for each class, and can be acquired by consuming "Skill Points (SP)". Skill points can be earned by clearing cocoons and towers. For the current number of skill points that can be earned, see Obtaining Skill Points on the Trainier page. The acquired skill points are common to all classes, and each point acquired will add 1 point to all classes. For details on class skills, see each class page. With the update on June 7, 2023, you can now freely reassign skill points. Specifications before June 7, 2023 Specifications before June 7, 2023 To reassign skill points, you had to do the following: Use "N-Initialize all skill trees" Add a skill tree with AC and transfer skill trees with SG Add skill tree Class counter: 500AC Add a skill tree to a specific class. Skill tree transfer Class counter: 100SG Move the added tree to another class. Transfer is only possible for classes that have two or more trees, and it is not possible to make a specific class without a tree. Points will be reset for the relocated tree. If Tree 1 and Tree 2 exist and you transfer Tree 1 to another class, Tree 2 will be moved up and become Tree 1. In-game notation regarding class skills Some class skills have certain conditions such as class composition and weapon type used in order to exert their effects. For example, if there is a description such as "main class/sword only", this indicates that "the skill is valid only if the class is set as the main class and a sword is used." *Please note that in the above case, "/" has the meaning of "AND", not "OR". Class skills with an icon that says "SP0" can be learned without consuming skill points. An example is the Force or Tector class skill Unite Technique.

## Resta sign
Recovers your HP by 50% of its maximum value. Image Lv. 1 - - Edit common items for this class It is in a state of learning from the start of the game. active skill. Can be set in a weapon/subpalette and used. Maximum stock number is 10. You can recover the stock by picking up the object "Resta Sign" on the field. You can purchase one for 250 N-Meseta at the "Item Shop" of each ARCS base (Central City / Ritem City / Kvaris Camp / Stear Camp).

## Reverser sign
Revives nearby players who are unable to fight. Image Lv. 1 - - Edit common items for this class It is in a state of learning from the start of the game. active skill. Can be set in a weapon/subpalette and used. Maximum stock number is 5. You can recover the stock by picking up the object "Reverser Sign" on the field. You can purchase one for 500 N-Meseta at the "Item Shop" of each ARCS base (Central City / Ritem City / Kvaris Camp / Stear Camp).

## Photon Blast (PB)
Converts photons in space into powerful energy and unleashes it as a deadly blow. Once activated, it cannot be activated again until the recharge is complete. Image Lv. 1 - - Edit common items for this class It is in a state of learning from the start of the game. active skill. Can be set in a weapon/subpalette and used. The maximum stock number is 1, and the recast time is 120 seconds. Different actions will be performed depending on the type of weapon being used. If you acquire the corresponding style skill in "EX Style" described below, you can activate "After Blast (AB)" by pressing the button again after PB is activated. When AB is activated, the icon design changes to a dedicated one. For other details, see the Weapons/Armor page.

## Easy operation
Various actions are automatically performed depending on the distance to the enemy and the equipped weapon. Image Lv. 1 - - Edit common items for this class It is in a state of learning from the start of the game. active skill. Can be set in a weapon/subpalette and used.

## dive attack
Makes a sudden dive from the air and blows away the enemies hit by the attack. With quick movements, you can evade attacks from enemies. Image Lv. 1 - - Edit common items for this class It is in a state of learning from the start of the game. active skill. Can be set in a weapon/subpalette and used.

## M.A.R.S.
Accumulate energy in the battery over time, and when it is charged to 25% or more, you can summon M.A.R.S. If you press the button again while M.A.R.S. is equipped, it will be unequipped. Image Lv. 1 - - Edit common items for this class Functions will be unlocked by completing the main task (Chapter 2) "Receiving 'M.A.R.S.'". active skill. Can be set in a weapon/subpalette and used. For other details, please see the M.A.R.S. page.

## Fomergion
A combined attribute technique of fire and darkness. Activates by consuming the composite attribute technique gauge. Generates a huge sword of flame and darkness and mowing down the front. Image Lv. 1 - - Edit common items for this class Can only be used if you have learned the Force class skill "Unite Technique Fo" or the Tector class skill "Unite Technique Te" and have set that class as your main class. active skill. Can be set in a weapon/subpalette and used. For other details, see the Techniques/Complex Attributes page.

## addon skills
Implemented on August 3, 2022 Add-on skills are special skills that are different from class skills. You can create and set different add-on skills for each class, and all effects will occur at the same time regardless of the class you play. See the Add-on Skills page for details.
