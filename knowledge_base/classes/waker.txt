Basic Fighting Method Each Familiar has a preferred fighting distance. If you want to attack from a long distance, use Fredoran, and if you want to fight at close range, use Wolfer. Quince, which can be summoned with a weapon action, accumulates energy while blocking attacks. Once the gear gauge is at maximum, hold down Weapon Action and throw a quince to deal great damage to the enemy. We recommend fighting by actively aiming for the powerful class skill "Familiar Unison". Check the gear gauges of Fredoran and Wolfer, and use the missing Photon Arts. Once you get used to it, a marker will appear by hitting the familiar's attack multiple times. By holding down the normal attack button, you can detonate the marker, recover a large amount of PP, and also generate a continuous attack called ``Familiar Assist''. "Familiar Assist" also has the effect of accumulating the gear gauge of the last familiar you summoned using PA, so while "Familiar Assist" is in progress, it's a good idea to fight mainly with the other familiar. According to the purple guide that appears after the PA, A "linear drive" occurs when performing a normal attack. It can be moved forward or backward depending on the input direction. This is useful when switching between "Fredran" and "Wolfer" or when you want to adjust the distance between you and the enemy. Familiars do not have any breeding elements like the old PSO2 pets, and a fixed one is summoned from among three types depending on photon arts and weapon actions. Direct attacks using Tact are also possible. Unlike Summoner's Tact, which could not be given special abilities, Waker's Tact can be given special abilities. Overall, Waker is a new class that is very different from the old PSO2 Summoner. As previously announced, "Waker" is a class that is similar to "Summoner". Until now, the classes distributed in "NGS" were designed as classes that followed the specifications of the old classes in "PSO2". However, there were some parts that were arranged for "NGS", and some people were confused by the difference in this respect from the same class in "PSO2". Therefore, when we decided to create a completely new class that uses tact as a weapon but goes in a different direction from "Summoner", we were concerned that if we kept the same class name, more people would feel uncomfortable with the difference from the old "Summoner" than the previous class, so we devised a completely different name. Another reason is that we wanted to encourage those who weren't interested in PSO2's Summoner to learn about Waker as a new class without preconceived notions.
