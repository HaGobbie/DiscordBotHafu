# Seasonal Points Exchange Shop and Point Exchange Rewards (Part 1)

== Overview ==
The Seasonal Points Exchange Shop is an event-limited Exchange Shop that is only available during Seasonal Events.
Previous iterations of the of the Seasonal Points Exchange Shop can be viewed on this page , or can be individually viewed on each event's Event Page .

== Seasonal Points ==
Seasonal Points ( ) are obtained by completing Xiandy's Tasks and Event Limited Tasks which will reward the player with Seasonal Points. Enemies with the Season Mark will also drop Seasonal Points upon death.
